# TestCaseVersta
 Для запуска необходимо наличие Docker и Visual Studio
1. Откройте в Visual Studio файл "WebApplication2.sln";
2. Проверьте в обозревателе решений наличие Docker_Compose;
3. Проверьте что в параметрах запуска указаны "Release" и "Docker_Compose";
4. Нажмите запустить;
5. Клиентское приложение распологается по адресу: "http://localhost/", посмотреть Api запросы можно по адресу: "http://localhost:5000/swagger"
